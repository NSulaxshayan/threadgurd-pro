Advanced Cybersecurity Research

Developed at Sri Lanka Institute of Information Technology

Our research presents a comprehensive approach to enhancing enterprise network security by leveraging machine learning techniques and rule-based monitoring to identify and prevent various threats. The study addresses critical security challenges through intelligent, automated solutions.

  • Data Exfiltration Attack Detection using HMM-LSTM
  • Manual Process and Protocol Bypass Monitoring
  • Peripheral Device Data Leakage Detection
  • Network Vulnerability with Threat Intelligence

Hybrid Machine Learning Innovation

Our groundbreaking HMM-LSTM approach combines Hidden Markov Models with Long Short-Term Memory networks, achieving 99.5% detection accuracy for real-time threat identification and automated response capabilities.

Research Methodology

Our comprehensive four-component approach to enterprise network security leverages cutting-edge machine learning and intelligent monitoring systems

Data Exfiltration Detection

Advanced hybrid machine learning model combining HMM and LSTM networks for real-time detection of sophisticated exfiltration attempts including HTTP POST-based attacks and DNS tunneling with PCA-based feature reduction.

Protocol Bypass Monitoring

Intelligent rule-based monitoring system for detecting manual process execution and unauthorized protocol bypass activities through time-duration analysis, command detection, and real-time alerting mechanisms.

Peripheral Device Security

Machine learning classification system using supervised and unsupervised techniques to monitor and prevent data leakage through USB drives, external storage devices, and Bluetooth connections with automated response.

Threat Intelligence Integration

Network vulnerability detection enhanced with real-time threat intelligence feedback from CVE databases, automated risk prioritization, and SIEM integration for comprehensive security monitoring.

Machine learning has emerged as a crucial component in cybersecurity due to its ability to detect and prevent evolving cyber threats more effectively than traditional rule-based security solutions. Our research builds upon existing work in ML-based security mechanisms.

Research Contributions:
  • Advanced ML algorithms integration (SVM, Decision Trees, Deep Learning)
  • Unsupervised learning techniques for behavioral threat detection
  • Predictive network vulnerability identification and early-stage exploit detection
  • ML-based anomaly detection for comprehensive protocol monitoring
Challenges Addressed:
  • False positives and adversarial attack mitigation
  • Computational efficiency and scalability optimization
  • Model interpretability and transparency enhancement
